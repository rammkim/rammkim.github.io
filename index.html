01. 자기소개

저는 웹 퍼블리싱은 물론 프론트엔드까지 하기 위해 열심히 공부 중인
2년 4개월 차 웹 퍼블리셔입니다.
새로운 기술로 더 나은 사이트를 만들어가는 일이 즐겁습니다. 🙂

사용자 입장에서 생각하며 효율적인 UI/UX가 무엇일지 계속 고민하고,
웹 표준과 웹 접근성을 고려한 마크업을 하기 위해 노력합니다.
협업을 하며 일하면서도 좀 더 좋은 방법이 있을까 하는 고민을 가지고
많은 소통을 하며 좀 더 나은 방향성을 찾아갑니다.
모든 사람이 사용하기에 편리하고, 즐거운 사이트를 만드는 것이
제가 하는 일이라고 생각합니다.

사용자를 배려하는 사이트를 만들어 보람을 얻고,
더 나은 코드를 작성하기 위해 끊임없이 공부하며,
때로는 제 뜻처럼 안되는 상황에 본인을 자책하기도 하지만,
제 장점인 긍정적이고 금방 털어버리는 성격 덕분에 다시 도전하여
결국엔 이루고 그 성취감이 저에게는 큰 에너지가 됩니다. 🤗
  • HTML을 이용한 마크업
  • SCSS 전처리기 사용
  • Javascript로 인터렉션 구현
  • Webpack을 통한 최적화 빌드
  • SourceTree를 이용한 Git 협업
